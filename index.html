C'est quoi le web pour moi ?

Le web pour moi, c'est un moyen visuel de pouvoir avoir le résultat de son code de manière assez concrète

Je suis plus orienté Front-end de manière générale, mais j'ai quand même un peu de connaissances en Back-end

Ma relation avec les outils de versioning

En DUT on a la malchance de rencontrer SVN. C'est de mon côté puis en entreprise que j'ai découvert les joies de Git. J'ai surtout compris l'importance des bonnes pratiques comme les reviews, les noms de commit et même les conventions de reviews

Comment marche un site selon moi ?

Un site est composé de plusieurs couches, le Front-end, le Back-end et la base de données. Le Front-end est ce que l'utilisateur voit, le Back-end est ce qui permet de faire fonctionner le site et la base de données est ce qui permet de stocker les données
